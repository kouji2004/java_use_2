 <, >という記号は数学でも用いる大小比較の記号です。x < yはxがyより小さければtrue,
 大きければfalseを返します。x > yはその逆になります。また数学で同様に用いる≦, ≧という記号（以上や以下を表します）は、<=, >=と記述します。

class Main {
  public static void main(String[] args) {
    // 「4 + 2」と「6」を比較し、falseとなるようにしてください
    System.out.println(⓵);
    
    // 「4 + 2」と「6」を比較し、trueとなるようにしてください
    System.out.println(②);
    
  }
}



解答
⓵4+2>6
②4+2 <= 6
